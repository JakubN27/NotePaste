# Changelog

## 0.1.0

- Initial NotePaste prototype.
- Added a desktop-only Obsidian plugin that inserts captured images into the active note.
- Added a native macOS Continuity Camera companion app.
- Embedded plugin files in the companion app and added vault install/update scaffolding.
- Added local plugin receiver, one-time upload tokens, and attachment saving.
- Added unsigned macOS `.dmg` release packaging.
